# Day 6 – 10 AI Use Cases in Engineering

## 1. Resume Screening ⭐ (Internship Domain)
**Description:** AI system that analyzes resumes and matches them with job descriptions to shortlist candidates.  
**Capability:** Extraction + Text (LLM)  
**Input → Output:** Resume PDF + Job Description → Match score, matched skills, missing skills, recommendation

---

## 2. Automated Code Review (Internship Domain)
**Description:** AI reviews pull requests and suggests improvements or detects bugs.  
**Capability:** Code  
**Input → Output:** Source code / PR diff → Code review comments and bug suggestions

---

## 3. Bug Ticket Classification (Internship Domain)
**Description:** AI categorizes and prioritizes bug reports automatically.  
**Capability:** Text  
**Input → Output:** Bug description → Category + priority level

---

## 4. Meeting Notes Summarization
**Description:** AI converts long meeting transcripts into concise summaries.  
**Capability:** Text  
**Input → Output:** Transcript → Summary + action items

---

## 5. Invoice Data Extraction
**Description:** AI extracts structured data from invoices.  
**Capability:** Extraction  
**Input → Output:** Invoice PDF/image → Name, amount, date, invoice number

---

## 6. Manufacturing Defect Detection
**Description:** AI detects defects in product images from production lines.  
**Capability:** Vision  
**Input → Output:** Product image → Defect type / OK status

---

## 7. Predictive Maintenance
**Description:** AI predicts machine failures using sensor data.  
**Capability:** Agent  
**Input → Output:** Sensor logs → Failure prediction + maintenance suggestion

---

## 8. Customer Support Chatbot
**Description:** AI answers customer queries automatically.  
**Capability:** Text  
**Input → Output:** User query → Response / ticket creation

---

## 9. Security Log Monitoring
**Description:** AI detects suspicious activity in system logs.  
**Capability:** Agent  
**Input → Output:** System logs → Alert / anomaly report

---

## 10. API Documentation Generator
**Description:** AI generates API documentation from backend code.  
**Capability:** Code  
**Input → Output:** Source code → API docs with examples

---

# ⭐ Use Case I Would Build First

## Resume Screening

I would build this first because it solves a real-world hiring problem where recruiters manually filter large numbers of resumes. Automating this process using AI can save time and improve consistency in candidate selection. It is also a beginner-friendly LLM project that I can later enhance with smarter ranking and explanations.
