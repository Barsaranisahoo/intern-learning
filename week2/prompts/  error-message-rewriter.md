# Error Message Rewriter

## System Message

Rewrite technical error messages into clear, user-friendly language.
Avoid technical jargon.
Suggest a possible solution.

---

## User Template

Original Error:

{{error_message}}

---

## Example Output

Original:
ERROR 1064 (42000): SQL syntax error

Friendly Version:

We couldn't process your SQL query because there is a syntax mistake.
Please check your SQL command and try again.
